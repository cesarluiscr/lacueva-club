# Club Campestre La Cueva - Sitio Web

Sitio web profesional para el Club Campestre La Cueva (Alajuela, Costa Rica).

## 📋 Características

- 🏊 Presentación de instalaciones
- 📅 Sistema de reservas en línea
- 👥 Portal de socios con acceso restringido
- 🛒 Tienda en línea (membresías)
- 💳 Integración de pagos (Stripe/PayPal)
- 📊 Panel administrativo
- 📱 Diseño responsive (mobile-first)

## 🚀 Stack Tecnológico

### Frontend
- React 18
- TypeScript
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- PostgreSQL
- JWT Authentication
- Stripe API

## 📁 Estructura del Proyecto

```
lacueva-club/
├── frontend/              # Aplicación React
├── backend/               # API Node.js/Express
├── database/              # Scripts de BD
├── docs/                  # Documentación
└── README.md
```

## 🛠️ Instalación y Setup

### Requisitos Previos
- Node.js v16+
- PostgreSQL v12+
- Git

### Inicio Rápido

```bash
# Clonar repositorio
git clone <repo-url>
cd lacueva-club

# Setup Frontend
cd frontend
npm install
npm run dev

# Setup Backend (en otra terminal)
cd ../backend
npm install
npm run dev
```
